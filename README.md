Ce projet concerne la refonte moderne du site institutionnel **Eden France Consulting**, développée avec **React** et **Vite**. L'objectif est de proposer une interface utilisateur plus dynamique, responsive et esthétique.

## Pour commencer

Pour lancer le projet sur votre machine locale, suivez les étapes ci-dessous :

### Prérequis

Assurez-vous d'avoir **Node.js** installé sur votre machine.

### Installation

Installez les dépendances du projet :

```bash
npm install
```

### Lancer le serveur de développement

Pour démarrer l'application en mode développement avec rechargement automatique :

```bash
npm run dev
```

Une fois la commande lancée, ouvrez votre navigateur à l'adresse du port indiquée sur votre console.

## Commandes Utiles

- `npm run dev` : Lance le serveur de développement.
- `npm run build` : Compile l'application pour la production.
- `npm run preview` : Prévisualise la version de production localement.
